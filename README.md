Features
Modern Design: Sleek, dark-mode aesthetic with custom animations and interactive elements.

Responsive: Built with a "mobile-first" approach ensuring a flawless experience on all devices.

High Performance: Optimized for speed, ensuring your clients get a fast-loading experience.

Conversion-Focused: Includes clear Call-to-Actions (CTAs), service breakdowns, and a functional contact form with validation.

Integrated Communication: Features a direct booking button(that show the booking form) and an EmailJS-powered contact form for seamless lead management.

Easy to Manage: Built with standard HTML, CSS, and JavaScript—no heavy frameworks, making it easy to customize and deploy.

🛠 Tech Stack
Frontend: HTML5, CSS3 (with custom variables and CSS Grid/Flexbox), and Vanilla JavaScript.

Styling: Custom CSS animations, transitions, and a "noise" texture overlay for a unique visual flair.

Integrations: * EmailJS/Cal.com/Setmore: For handling contact form submissions directly to your email.

Google Fonts: Utilizing Playfair Display and Syne for a professional typographic hierarchy.

📁 Project Structure
index.html: The main structure of the portfolio site.

css/styles.css: Contains all styles, animations, variables, and responsive media queries.

js/script.js: Handles interactivity, form validation, scroll animations, and external integrations (like booking links and email services).
